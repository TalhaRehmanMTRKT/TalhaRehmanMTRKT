What is Linear Programming?

Linear programming is a mathematical method for finding the best outcome in a model whose requirements are represented by linear relationships. The general form of an LP is:

$$ \min_{x} \quad c^T x $$ $$ \text{subject to:} \quad Ax \leq b, \quad x \geq 0 $$

Where $c$ is the cost vector, $A$ is the constraint matrix, $b$ is the right-hand side vector, and $x$ is the vector of decision variables.

Setting Up CPLEX in C++

Before writing the optimization model, ensure that IBM CPLEX is installed and that your IDE is configured with the correct include and library paths.

A Minimal Example

Let's solve a simple problem:

$$ \min \quad x_1 + 2x_2 $$ $$ \text{s.t.} \quad x_1 + x_2 \geq 10, \quad x_1 \leq 8, \quad x_1, x_2 \geq 0 $$ 
#include <ilcplex/ilocplex.h>
#include <iostream>

int main() {
    IloEnv env;
    try {
        IloModel model(env);

        // Decision variables
        IloNumVar x1(env, 0, IloInfinity, ILOFLOAT, "x1");
        IloNumVar x2(env, 0, IloInfinity, ILOFLOAT, "x2");

        // Objective: minimize x1 + 2*x2
        model.add(IloMinimize(env, x1 + 2 * x2));

        // Constraints
        model.add(x1 + x2 >= 10);
        model.add(x1 <= 8);

        // Solve
        IloCplex cplex(model);
        cplex.solve();

        std::cout << "Optimal x1 = " << cplex.getValue(x1) << std::endl;
        std::cout << "Optimal x2 = " << cplex.getValue(x2) << std::endl;
        std::cout << "Objective   = " << cplex.getObjValue() << std::endl;

    } catch (IloException& e) {
        std::cerr << "CPLEX Error: " << e.getMessage() << std::endl;
    }
    env.end();
    return 0;
}

Understanding the Output

Running this program yields $x_1 = 8$ and $x_2 = 2$, with an optimal objective value of $8 + 2 \times 2 = 12$. The solver verified that this solution is globally optimal.

Tip: Always check the CPLEX solution status using cplex.getStatus() to confirm the solver found an optimal solution before using the results.

Key Concepts

  • Decision Variables: The unknowns you're solving for ($x_1$, $x_2$).
  • Objective Function: What you want to minimize or maximize.
  • Constraints: The conditions the solution must satisfy.
  • Feasible Region: The set of all points satisfying all constraints.
